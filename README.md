\# Employee Management System



A Django-based Employee Management System to manage employee records.



\## Features



\- Add employee details

\- View employee list

\- Edit employee information

\- Delete employee records

\- Employee detail page

\- User login authentication



\## Technologies Used



\- Python

\- Django

\- HTML

\- CSS

\- SQLite

\- Git \& GitHub



\## Project Structure



\- employee\_management - Django project folder

\- employees - Employee management application

\- manage.py - Django management file



\## How to Run



1\. Clone the repository



git clone https://github.com/Sachinvelmurugan/EmployeeManagement.git



2\. Go to project folder



cd EmployeeManagement



3\. Create virtual environment



python -m venv venv



4\. Activate virtual environment



venv\\Scripts\\activate



5\. Run server



python manage.py runserver
